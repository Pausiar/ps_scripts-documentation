🚀 Guía de Instalación

1

Descargar y Extraer

Descarga el script y extráelo en tu carpeta de recursos

# Extraer en la carpeta de recursos
[esx]/ps_rental/
2

Configurar Base de Datos

Importa el archivo SQL para crear las tablas necesarias

# Ejecutar en tu base de datos MySQL
source rentvehs.sql
3

Configurar Servidor

Añade el script a tu server.cfg

# Añadir al server.cfg
ensure ps_rental
4

Configurar Compatibilidad

Edita config.lua para que coincida con tu configuración de servidor

Config.Compatibility = {
    ['fuel_system'] = 'cdn-fuel',      -- Tu sistema de combustible
    ['key_system'] = 'vehiclekeys',    -- Tu sistema de llaves
    ['banking_system'] = 'esx',        -- Sistema bancario
    ['notification_system'] = 'esx',   -- Sistema de notificaciones
}
5

Configurar Administradores

Define los grupos de administradores en la configuración

Config.AdminGroups = {
    'admin',
    'superadmin',
    'owner'
}

⚙️ Configuración Avanzada

Configuración Básica

Config.Settings = {
    ['max_rental_days'] = 30,          -- Días máximos de alquiler
    ['min_rental_days'] = 1,           -- Días mínimos de alquiler
    ['cleanup_interval'] = 3600000,    -- Intervalo de limpieza (1 hora)
    ['default_fuel'] = 100,            -- Nivel de combustible por defecto
    ['enable_notifications'] = true,   -- Activar notificaciones
    ['enable_blips'] = true,           -- Mostrar blips en el mapa
    ['enable_target'] = true,          -- Integración ox_target
}

Personalización de UI

Config.UI = {
    ['search_enabled'] = true,         -- Habilitar búsqueda de vehículos
    ['categories_enabled'] = true,     -- Habilitar filtrado por categorías
    ['image_fallback'] = true,         -- Usar imágenes de respaldo
    ['theme'] = 'dark',                -- Tema de la UI (dark/light)
    ['currency_symbol'] = ',         -- Símbolo de moneda
    ['date_format'] = 'DD/MM/YYYY',    -- Formato de fecha
}

🎮 Guía de Uso

👥

Para Jugadores

  • Buscar ubicaciones de alquiler en el mapa
  • Acercarse al NPC de alquiler
  • Usar ox_target para abrir el menú
  • Navegar por categorías y buscar vehículos
  • Seleccionar duración del alquiler (1-30 días)
  • Elegir método de pago (efectivo/tarjeta)
  • Completar el alquiler
👨‍💼

Para Administradores

  • Usar /rentveh para abrir panel admin
  • Gestionar vehículos en tiempo real
  • Crear nuevas ubicaciones de alquiler
  • Obtener coordenadas automáticamente
  • Teletransportarse a ubicaciones
  • Monitorear estadísticas del sistema

Comandos Disponibles

# Abrir panel de administración (solo admins)
/rentveh

# Listar configuración actual (solo admins)
/rentveh_list

🗄️ Estructura de Base de Datos

rental_vehicles

Catálogo de vehículos con categorías y precios

  • ID, categoría, nombre
  • Precio por día
  • URL de imagen personalizada

rental_locations

Ubicaciones de alquiler y configuración

  • Coordenadas del NPC
  • Punto de spawn de vehículos
  • Modelo de NPC asignado

rentvehs

Seguimiento de alquileres activos

  • ID del jugador y vehículo
  • Matrícula del vehículo
  • Tiempo de expiración

owned_vehicles

Integración con sistema ESX

  • Propiedad temporal de vehículos
  • Integración con garajes ESX
  • Limpieza automática

🔧 Solución de Problemas

⚠️

NPCs no aparecen

  • Verificar instalación de ox_target
  • Comprobar conexión a base de datos
  • Revisar configuración de ubicaciones
🚗

Vehículos no spawean

  • Verificar que no hay obstáculos en el spawn
  • Comprobar que los modelos de vehículos existen
  • Revisar integración del sistema de combustible
👨‍💼

Panel admin no abre

  • Verificar configuración de grupos admin
  • Comprobar permisos del jugador
  • Revisar console por errores

Modo Debug

Activa el modo debug para obtener información detallada:

Config.Debug = {
    ['enable_logging'] = true,         -- Activar logging en consola
    ['log_level'] = 'debug',          -- Nivel de log
    ['enable_admin_logs'] = true,     -- Log de acciones admin
}
