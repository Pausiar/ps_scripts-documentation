Guía de Instalación

Instalación Rápida
Descargar y Extraer

Descarga la última versión y extrae en tu carpeta de resources

resources/[admin]/ps_admin/
Configuración de Base de Datos

Importa el archivo SQL para crear las tablas necesarias

mysql -u usuario -p nombre_base_datos < database_tables.sql
Configuración

Edita config.lua para que coincida con la configuración de tu servidor

Config.Framework = 'esx' -- o 'qb-core'
Config.AdminCommand = 'admin'
Config.QuickMenuKey = 'F4'
Configuración de Permisos

Configura los permisos ACE en tu server.cfg

add_ace group.admin admin.* allow
add_principal identifier.steam:TU_STEAM_HEX group.admin
Iniciar Recurso

Añade a tu server.cfg e inicia

ensure ps_admin

Referencia de API

Exports del Servidor
-- Verificar si el jugador es admin
exports['ps_admin']:IsPlayerAdmin(source)

-- Banear un jugador
exports['ps_admin']:BanPlayer(source, reason, duration)

-- Crear un evento
exports['ps_admin']:CreateEvent(adminSource, eventData)
Exports del Cliente
-- Abrir menú de admin
exports['ps_admin']:OpenAdminMenu()

-- Verificar estado noclip
exports['ps_admin']:IsNoclipEnabled()

-- Mostrar notificación
exports['ps_admin']:ShowNotification(message, type)
Eventos
-- Escuchar acciones de admin
AddEventHandler('PSAdmin:PlayerBanned', function(data)
    print('Jugador baneado:', data.playerName)
end)

-- Activar eventos personalizados
TriggerEvent('PSAdmin:CustomAction', data)
